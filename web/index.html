💼 SecureBank Console App

This Java project simulates a secure, real-world banking environment through a command-line interface.

🔑 Features

  • User Registration and Login with file-based persistence
  • Deposit, Withdraw, Transfer between accounts
  • Transaction logging and balance tracking
  • Modular, scalable, and error-tolerant design

🗂️ Folder Structure

ConsoleBankingSystem/
├── src/
│   ├── model/
│   ├── service/
│   ├── util/
│   └── Main.java
├── data/
│   ├── users.txt
│   ├── transactions.txt
│   └── accounts/
├── web/
│   ├── index.html
│   └── style.css
└── compile_run.bat

📦 How to Run

  1. Ensure javac is installed and set in PATH
  2. Open terminal and run compile_run.bat (for Windows)
  3. Use CLI to register, login, and simulate transactions
